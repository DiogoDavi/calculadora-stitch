# Calculadora Stitch | HTML • CSS • JavaScript
Calculadora web responsiva com tema inspirado no Stitch, desenvolvida com HTML, CSS e JavaScript puro.
## Funcionalidades
- Operações básicas (+ − × ÷)
- Botão limpar (C)
- Layout responsivo (mobile, tablet e desktop)
- Interface temática do Stitch
